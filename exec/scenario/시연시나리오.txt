시연 시나리오
- 모든 알람 끄기
- 알러지 있을 때 진동은 보여줘야 하니까 진동 켜기
- 방해금지 모드 켜기
- 위시리스트 이미지 업로드 용 제품사진 확보
- 화면 출력 테스트 및 방송 테스트(특히 디스코드 사용시 유의)
- 우리가 카메라 실행하는 부분이 있으니까 방송 송출이 제대로 되는지 확인해봐야할듯?
- 기타 필요사항 시연 고민하면서 적어보기!


*기본적으로는 내가 할 대사가 적혀있고 (시연) 적혀있는 부분이 형이 할 행동인데 궁금한 점은 물어봐줘! 그냥 적당히 흘러가는 대로 해도 될듯ㅋㅋㅋ 내일 좀 일찍 갈 테니까(8시 30분쯤) 호흡 한 번 맞춰봅시다


# 1. 개인정보 동의
- (시연) 회원 정보 삭제하고(사전에 확인) 어플 실행
- 어플리케이션을 시작하면 개인정보 동의 제공을 묻는 알림창 등장
- 모바일 기기의 device id를 받아와서 유저 회원 가입을 처리
- 외국인 여행객을 위한 일회성 서비스이므로 회원가입 과정을 간소화 하는 것이 편리할 것 같아 정보 입력을 최소화하였습니다
- (시연) 동의 버튼 클릭

# 2. 비선호 재료 선택
- 개인적으로 선호하지 않거나 알러지가 있는 재료를 선택합니다.
- (시연) 갑각류, 바나나,  오징어 선택
- 이렇게 선택을 마치고,
- (시연) Next 버튼 클릭:  Next 버튼을 누르면 회원가입이 완료됩니다.

# 3. 홈 화면
3-1. 테마별로 추천 제품을 보여줌
3-2. 현재 한국에서 조회수 기반 상위 10개의 제품을 보여줌 (시연: 가로 스크롤)
3-3. 유저의 히스토리를 기반으로 추천 제품을 보여줌 (시연: 가로 스크롤)
      
# 4. 에디터 추천(불닭)
- (시연) 상단 까르보불닭볶음면 이미지 클릭
- (시연) 아래로 스크롤: 까르보불닭 제품의 상세 설명 소개
- 하단 까르보불닭볶음면과 연관 제품 소개 (시연: 가로 스크롤)

# 5. 검색
- (시연) 네비게이션 바 '검색' 클릭: 일별 조회수가 가장 높은 카테고리를 보여줍니다
- (시연) 일일랭킹 '컵라면' 클릭: 랭킹 3위인 컵라면을 클릭하면 컵라면 카테고리의 제품들을 보여줍니다.
- (시연) 전체 카테고리 검색(All 버튼 클릭): 만약, 특정 제품을 찾고 싶다면 카테고리를 통해 제품을 검색할 수 있도록 구현하였습니다.
- (시연) 맨 위 스낵/초콜릿/씨리얼 -> 파이 클릭: 이렇게 카테고리 검색을 통해 사용자는 원하는 종류의 제품을 찾을 수 있습니다.
- (시연) 몽쉘 제품 클릭 

# 6. 카메라
- 다음으로는 저희 서비스의 핵심 기능인 인공지능 스캔을 소개하겠습니다.
- (시연) 네비게이션 바 '카메라' 클릭
- (시연) 사진 기반 스캔 '바나나킥'스캔
제품 성분에 회원가입 과정에서 선호하지 않는 재료가 포함된 경우 팝업 알림과 진동을 통해 사용자가 이를 명확하게 인지할 수 있도록 하였습니다. 뿐만 아니라 인공지능에 사진이 없는 데이터가 있을 경우에도 제품 정보를 확인할 수 있도록 저희는 바코드 기반 스캔 기능도 추가하였습니다.
- (시연) 카메라 실행 후 기다렸다가 바코드 화면 클릭
- (시연) 바코드 기반 스캔 '닭다리' 스캔 (6247개 제품)
사진이 등록되지 않은 제품일지라도 바코드 스캔을 통해 제품을 검색할 수 있게 함으로써 서비스의 완성도를 높이고자 하였습니다.

# 7. 위시리스트
미리 제품 사진 저장해둬야함!!! 
- (시연) 네비게이션 바 '위시리스트' 클릭
- (시연) 이미지 업로드 버튼 클릭
- (시연) 오징어집, 홈런볼 사진 업로드
- (시연) 모델이 인식해서 사진 뜨면 사진 클릭해서 상품 디테일 페이지로 이동 가능 ( 홈런볼만 디테일 들어가서 위시리스트 추가 )
- (시연) 그리고 돌아와서 Correct 버튼 클릭하면 위시리스트 추가 완료
- (시연) 이미지 편집 버튼 클릭 - 구매처리 / 구매취소처리 / 위시리스트 삭제 가능

# 8. 마이페이지
- (시연) 네비게이션 바 '마이페이지' 클릭
- (누르지는 말고 화면에서 대기) 회원탈퇴 가능합니다
- (시연) 비선호 재료 편집
- (누르지는 말고 화면에서 대기) 히스토리도 조회 가능합니다

# 9. 마무리
- 이것으로 서비스 시연을 마치도록 하겠습니다.
